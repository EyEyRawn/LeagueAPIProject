FAQ

  • What is this website?

    If you stumbled across this by accident, this is a project designed for a college course. Specifically, it's an API website that will look up the average stats of a League of Legends player's last 20 games and display them. An important thing to know is that as of now, the website only works in Google Chrome with CORS disabled.

  • Why is the website so small?

    There's two main reasons for this. The first is that the website uses a development API key, which severly limits the number of requests I can make for information. The second is that this is a big project for me, I made it by myself, and I've never done anything like this before!

  • How does it work?

    There's two main features. First is the profile page, which lets you view the results of a single profile. Second is the versus page, which lets you look at two different profiles at the same time to compare their stats. See who's doing better, you or your friends! To use either though, you need to disable CORS in your browser.

  • What is CORS, and how do I disable it?

    CORS is a browser security feature. While handy, it blocks API requests unless whitelisted. When using this website you'll need to disable it, as it uses a limited API key. There are extensions for popular browsers that support this. Try using this extension for Google Chrome. As a best practice, reenable CORS when you're done; it's not a one-stop-shop for safety, but it is one of many active security features in your browser to give you a safe experience on the web. Every bit of help counts!

  • Why won't my stats show, I know I entered the right name!

    Assuming you've disabled CORS, this is because one of two reasons. Either your profile isn't on an American server, or you don't have a unique username (anything ending in #NA1). At the moment I only support American profiles that have a unique username. Profiles with a non-unique username (Randomdude#BARK) won't be found.

  • Why does it take so long to show my stats?

    The information displayed is gathered from the Riot Games API so it may take a few seconds for the information to load. There's no database, so the information is gathered each time you submit a username. Please allow three to five seconds for the information to load.

  • The website was working, but now it's not, what gives?

    The limited API key I have access to puts a limit on the number of requests. At the moment it will allow you to search for or update a profile about four times every two minutes. If you aren't making multiple requests, it's probably because I haven't updated the API key yet (it needs to be replaced with a new key given to me every 24 hours).
