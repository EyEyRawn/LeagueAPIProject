FAQ

  • What is this website?

    If you stumbled across this by accident, this is a project designed for a college course. Specifically, it's an API website that will look up the average stats of a League of Legends player's last 20 games and display them.At this point the API key has expired so this website is no longer functional.

  • Why is the website so small?

    There's two main reasons for this. The first is that the website uses a development API key, which severly limits the number of requests I can make for information. The second is that this is a big project for me, I made it by myself, and I've never done anything like this before!

  • How does it work?

    There's two main features. First is the profile page, which lets you view the results of a single profile. Second is the versus page, which lets you look at two different profiles at the same time to compare their stats. See who's doing better, you or your friends!

  • Why won't my stats show, I know I entered the right name!

    This is because one of two reasons. Either your profile isn't on an American server, or you don't have a unique username (anything ending in #NA1). At the moment I only support American profiles that have a unique username. Profiles with a non-unique username (Randomdude#BARK) won't be found.

  • Why does it take so long to show my stats?

    The information displayed is gathered from the Riot Games API so it may take a few seconds for the information to load. There's no database, so the information is gathered each time you submit a username. Please allow three to five seconds for the information to load.
